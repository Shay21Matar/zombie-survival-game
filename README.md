# zombie-survival-game
Zombie Survival Game

A zombie survival game developed as a programming project, where the player must survive waves of zombies by managing health, movement, and game mechanics. The project focuses on clean code structure, game logic, and basic testing.

# Game Overview

In this game, the player:

Navigates a survival environment

Encounters zombies that reduce player health

Must survive as long as possible without health reaching zero

Loses the game when the player’s health is depleted

The project emphasizes object-oriented design, game state management, and modular code organization.

# Technologies Used

Language: C++

Framework / Tools: Qt Creator

Version Control: Git & GitHub
